## Findings considered and rejected

- **`sudo` deprecation (sudo-rs/approx)**: NixOS unstable still defaults to `sudo`; not urgent, and switching is a policy decision, not a code fix. Not worth a plan.
- **`imports` inside `config = lib.mkIf ...` in `modules/nesw.nix`**: the "disabled module" pattern — unusual but a known working NixOS idiom. Not a bug.
- **`Notch.qml` audio-state timers possibly leaving `expanded=true`**: traced the timer paths; `audioTimer` collapses the notch when not hovering. No clear bug; would need runtime repro. Not planned.
- **Performance of `Launcher.qml` `results` re-scan / `Notch.qml` occupied rebuild**: app/workspace counts are tiny; no measurable cost. Not worth a plan.
- **`drivers/default.nix` `with lib;`**: idiomatic for this repo's Nix style; not debt.
- **Adding Nix formatter (alejandra/nixfmt) to CI**: no formatter is configured in the repo; introducing one would reformat the entire tree (large, opinionated diff). Out of scope; 004 adds only the already-configured `stylua` gate.
